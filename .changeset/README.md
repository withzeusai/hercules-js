# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the main changesets repo](https://github.com/changesets/changesets).

**Changesets workflow:**

1. Make your changes
2. Run `pnpm changeset` to create a changeset describing what changed
3. Commit the changeset file along with your code changes
4. When ready to release, run `pnpm changeset:version` to bump versions and update changelogs
5. Run `pnpm changeset:publish` to publish to npm
